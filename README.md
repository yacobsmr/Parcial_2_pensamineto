# ANALISIS

# Punto 1
el programa calcula el Máximo Común Divisor de dos números primero usa un filtro de seguridad que obliga al usuario a ingresar únicamente números enteros positivos repitiendo la pregunta si detecta letras o valores inválidos hasta q lo haga, una vez obtiene los dos números correctos aplica el algoritmo de Euclides el cual divide los números y reemplaza los valores con sus residuos repetidamente hasta que la división sea exacta osea que el residuo sea cero y ese último número utilizado es el resultado final el cual el programa organiza y muestra en pantalla. la estructura se divide en tareas específicas evita que el sistema colapse si alguien se equivoca al escribir y hace que todo el proceso sea organizado.

# Punto 2
el programa calcula una aproximación de una función matemática compleja el cual es el valor de e elevado a la potencia, utilizando un método de la serie de Taylor. el código trabaja en equipo usando funciones separadas, primero incluye un codigo que calcula factoriales multiplicando secuencias de números de forma progresiva y despues utiliza esos resultados dentro de otro ciclo que suma fracciones matemáticas paso a paso, logrando un resultado más exacto entre más operaciones se le indique realizar y por ultimo un bloque que maneja todo el proceso pidiendole a el usurio que ingrese el número a calcular y el nivel de precisión que desea y despues poner a trabajar las fórmulas y mostrar la respuesta final en pantalla
